黃金世紀的豐盛療癒（複訓）

Jun 08, 2025
閱讀時間: 1 分鐘

  • 專注度影響能量的程度跟深度，以及創造東西的強度
  • 情緒上的豐盛：能感覺到感謝或愛；意識上的豐盛：感受到自我價值、對自己在做的事感到榮耀
  • 帶著服務的心去工作，金錢才會隨之而來
  • 提供服務的價值以專注的深度與能量來決定，不犧牲自己
  • 花光積蓄沒有比自我內耗還可怕
  • 黃金世紀的法則是透過服務來賺錢
  • 認同這件事情是事實（接納自己的現況），才有力量去改變（讓自己更好去改變現況）

意識層級創造豐盛

  • 覺察限制性信念：
  1. 認為不夠（金錢不夠讓我花、我不夠好）-> 右肩、腹部 / 喉嚨、左肩
  2. 認為自己不值得 -> 喉嚨、左肩
  3. 生存的恐懼信念（害怕沒錢、要穩定工作）-> 胸口 / 額頭、左大腿
  4. 金錢的制約信念（工作才能賺錢）-> 雙腿的膝蓋

  • KALI女神的黑光鑽石火焰冥想：將限制性信念丟進火焰燃燒淨化，丟出去之後感覺對應的身體部位有變輕一點，後面整個火焰在身體燃燒時覺得比較熱，KALI女神說「你什麼都不用做，就能夠活得很好」

  • 豐盛祈禱文：

  1. 我的金錢源源不絕地湧出，花也花不完
  2. 我一直都足夠好也足夠完美
  3. 我值得豐盛與美好四面八方向我湧來
  4. 我擁有足夠的金錢，想要什麼都能輕易的入手
  5. 我可以輕鬆自在的賺取每一分錢
  6. 我全然的放鬆與信任宇宙的指引去喜悅的生活
  • 可以每晚睡前先淨化火焰冥想後唸禱詞

在能量層級創造豐盛

  • 金光蓮花聖殿的豐盛療癒：這是一個很感人的冥想，光是一開始的Mentra就有點眼眶濕了，進入金光蓮花聖殿後，看到純白的聖殿上面有一朵非常大又閃耀著光芒的金色蓮花，當聽到聖哲曼大師要將這朵蓮花放入內心時，我還很驚訝的想說這麼大朵放的進去嗎？結果就真的用很緩慢的速度慢慢融進了我的心中，在體內緩緩的綻放光芒；接著往後走到花園，我看見方型的花園，四周有許多的蓮花漂浮著，休息了一會兒；最後大朵蓮花的花瓣綻放成一道道門，代表我們的每一個須被憶起的力量，我看見門是以菱形的方式排列著，前面、左邊、右邊、後面的門比較大，中間分別各有兩扇小門左右，我選擇先進了我面前那道門，在一進去還沒問是什麼力量時，我就知道是勇氣了，就開始哭了，再進去左邊那道門，裡面是自信，這時就哭更多了…完全是我現階段很需要的力量沒錯，我感受到勇氣化為體內一道黃色+紅色的火焰在燃燒綻放，自信化為背後支撐我的白色羽翼，很感動我能重新領回這兩個力量，在需要的時候喚醒他們支持我、協助我。
  • 結果豐盛療癒完後就瘋狂拉肚子orz

在身體和物質層級創造豐盛

  • 讓自己豐盛的食物和習慣（冥想詢問高我）：
  1. 高我名字是迷蝶
  2. 食物：櫻桃、小番茄、高麗菜
  3. 習慣：運動（最好每天）

顯化願望冥想-神聖的銀白龍光

  • 財神：虎爺、蓋婭、默基瑟德
  • 分別給予我的願景力量、支持、智慧
  • 家中的豐盛聖壇上擺水晶、鮮花、自己用能量製成的東西（比如有水晶光之母的水晶也算、有神聖母親能量的噴霧也算）

心得

在課堂一開始就有好幾句打中我，完全符合我現在的狀態，好幾次都被感動的很想哭啊QQ

在講到“專注度影響能量的程度跟深度”時，我忽然發現原來我這麼喜歡花晶療癒手法的原因就在這，因為它的重點不是任何的技巧，而是專注一專注在自己、也專注在對方，這個專注的服務意外的很合我胃口，也沒想到這才是服務的關鍵，這更堅定我繼續鑽研在這條路上的決心了。 接著第二句打中我的是“花光積蓄沒有比自我內耗還可怕”…我的確年初離職到現在，幾乎都是在用積蓄生活，只是我仍然照樣吃飯、買需要的東西（包含花晶）、上想上的課，沒有過的比較拮据，但我這幾天忍不住有冒出一點是不是該省點錢或是該趕快找工作賺錢的想法，然後就被這句喚醒了XD 確實沒有比自我破壞更可怕的了，畢竟錢再賺就有，自我破壞的影響反而是更大更深的…

金光蓮花聖殿的豐盛療癒也是讓我覺得很感人的一段，尤其是找回勇氣和自信時，讓我發現我有多需要這兩個力量，看著其他還沒開的門，就知道仍有許多力量還待我之後療癒去憶起，好期待呀！…然後做完冥想就開始瘋狂拉肚子排毒了，第一次遇到來這麼快的哈哈哈，只是可以不要在上課時嗎XD

最後在種種子的部分，覺得真的很容易時間過了就會忘記，尤其有些好種子其實很習慣不自覺地會去種，但卻很少記得要迴向，練習有意識的種種子、有意識的迴向真的是太需要被提醒了，也突然發現如果跟療癒圈的朋友同學互相支持的話，那會是一個多好的種子跟循環，應該多多彼此給出服務，這樣也可以一起豐盛！

隔了兩年，這次的種子計畫設定在找到好工作上，但這個工作仍然是為了讓我可以更穩定的給出療癒服務，與兩年前目標類似，只是原本是增加數量，現在我更希望能在療癒中給出愛與支持，原本早上在讀價格時也是各種懷疑，但神聖母親一句“你只需要專注的給出愛”就讓我感動不已，意識到我的目的性及保有我的初衷才是最重要的，其他都不重要。
